# PWDConnect PH - Entire System ZIP

This package contains the complete integrated system for:

PWDConnect PH: Services and Benefits Portal for Persons with Disabilities

Included modules:
- Login
- Registration
- Logout
- User Dashboard
- Admin Dashboard
- Browse Services
- Apply for Services
- Track Applications
- Announcements
- Support Requests
- Profile Management
- Backend API
- SQLite Database
- SQL database schema
- Project documentation PDF

## Default Admin

username: admin
password: admin12345

## Run Backend

cd backend
npm install
npm run dev

## Run Frontend

cd frontend
npm install
npm run dev
